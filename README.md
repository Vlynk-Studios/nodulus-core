# Nodulus Suite

Welcome to the Nodulus monorepo. This repository contains the core framework and its companion packages.

## Structure

- [`packages/core`](./packages/core): The main Nodulus framework.
- [`packages/eslint-plugin-nodulus`](./packages/eslint-plugin-nodulus): Architecture validation rules for ESLint.

## Development

### Prerequisites

- Node.js >= 20.6.0
- NPM

### Installation

```bash
npm install
```

### Build Everything

```bash
npm run build
```

### Run Tests

```bash
npm test
```

### Linting

```bash
npm run lint
```
